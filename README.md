# Gerador de Senhas

Um programa simples que gera senhas aleatórias seguras, permitindo ao usuário escolher o tamanho e os tipos de caracteres que deseja incluir.

## Funções
- Definir o tamanho da senha.
- Escolher caracteres como:
  - Letras maiúsculas
  - Letras minúsculas
  - Números
  - Símbolos especiais
